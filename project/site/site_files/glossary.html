📖 Глоссарий

Словарь терминов, используемых в проекте PC Builder

ПК

Персональный компьютер; вычислительная машина, собранная из отдельных компонентов (процессор, видеокарта, память и т.д.) для решения задач пользователя.

Гайд

Пошаговое руководство или инструкция (текстовая или видео), помогающая пользователю разобраться в выборе комплектующих, процессе сборки или настройке системы.

Комплектующие

Отдельные аппаратные компоненты компьютера (процессор, материнская плата, видеокарта, оперативная память, накопители, блок питания, корпус), которые подбираются вместе для создания единой системы.

Сайт

Веб-ресурс (интернет-магазин), предоставляющий пользователям инструменты для подбора конфигурации, проверки совместимости, заказа деталей и оформления услуг по сборке.

ПО

Программное обеспечение; совокупность программ, установленных на компьютер для управления его работой и выполнения задач (операционная система, драйверы, утилиты, игры).

Глоссарий

Словарь узкоспециализированных терминов, используемых в проекте, с их краткими пояснениями для унификации понимания между командой и клиентами.

Админ панель

Закрытый раздел сайта, доступный только сотрудникам компании, предназначенный для управления товарами, заказами, пользователями, контентом и настройками конфигуратора.

Real-time (Реальное время)

Режим работы системы, при котором данные обновляются мгновенно. В контексте конфигуратора: изменение цены или статуса совместимости происходит сразу же после выбора любой детали, без перезагрузки страницы.

Сообщество

Группа пользователей проекта (клиенты, энтузиасты, эксперты), объединенных интересом к компьютерному железу, общающихся на форуме или в социальных сетях проекта.

Button_Switch_Styles

Элемент интерфейса (кнопка-переключатель), позволяющий пользователю менять визуальное оформление страницы (например, переключение между светлой и темной темой или разными стилями отображения карточек товаров).

Фреймворк

Программная платформа или каркас, на базе которого разработчики создают сайт и конфигуратор, ускоряя процесс разработки и обеспечивая стабильность кода (например, React, Vue, Laravel).

Форум

Раздел сайта или отдельная площадка для дискуссий, где пользователи могут задавать вопросы, делиться своими сборками, обсуждать новости железа и получать помощь от сообщества и модераторов.

Верстка сайта

Процесс перевода дизайна макета в код (HTML/CSS), определяющий визуальное отображение элементов страницы, их расположение, адаптивность под мобильные устройства и анимации.

Backend (Бэкенд)

«Серверная» часть сайта, скрытая от глаз пользователя. Отвечает за логику работы конфигуратора (проверку совместимости), хранение базы данных товаров, обработку платежей и формирование заказов.

Frontend (Фронтенд)

«Клиентская» часть сайта, с которой взаимодействует пользователь. Включает в себя интерфейс конфигуратора, каталог товаров, кнопок, форм ввода и визуальные эффекты.

Сборка

Услуга или физический процесс монтажа выбранных комплектующих в корпус, установки системы охлаждения, кабелей, а также первоначальной настройки ПО и тестирования системы в мастерской компании.

Стек технологий

Набор языков программирования, фреймворков, библиотек, баз данных и инструментов, используемых для создания и работы программного продукта.

Хост

Сервер (физический или виртуальный), где размещается проект.

LAMP Stack

Набор ПО для веба: Linux (ОС) + Apache (сервер) + MySQL (база) + PHP (язык).

Тайм-триггер

Инструмент для учета времени, затраченного на задачи (например, Toggl, Clockify).

Чек-лист

Список обязательных шагов для проверки готовности задачи.
